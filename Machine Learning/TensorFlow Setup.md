## TensorFlow 설치

### Python 설치 (Python 3.6~3.9)

```bash
python --version

py -3.9 -m venv tf-env # 여러 버전이 깔려있어, 버전을 지정해서 가상환경 생성

tf_env\Scripts\activate # 활성화  나올는 deactivate

pip install tensorflow

python -c "import tensorflow as tf; print(tf.__version__)" # 설치확인

```
